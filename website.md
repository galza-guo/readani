# readani

readani is a desktop bilingual PDF reader built for fast, focused reading and sentence-level translation. It pairs a crisp PDF view with an intelligent translation pane, so you can read naturally while keeping translations close and contextual. The interface is clean, English-only, and optimized for long reading sessions.

## One‑Line Pitch
Read PDFs and see sentence‑level translations side‑by‑side, instantly.

## Core Value
- Understand foreign‑language PDFs without switching tools or losing context.

## Target Users
- Researchers and students reading papers in another language
- Professionals reviewing multilingual reports and manuals
- Language learners who want sentence‑by‑sentence translation while reading

## Key Features
- Two‑column layout: PDF on the left, translations and controls on the right
- Sentence‑level alignment with precise highlights on the PDF
- Hover or click a sentence to highlight its location in the document
- Smart translation window by page range to keep context
- Read‑only rich text rendering for clean, distraction‑free translation output
- Local JSON cache for faster re‑reads and consistent results

## Differentiators
- Built for documents, not copy‑paste snippets
- Visual, sentence‑mapped highlights instead of block translations
- Minimal UI tuned for long‑form reading
